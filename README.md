# Oxford Advanced Calculator

A highly comprehensive, advanced calculator web application engineered explicitly for Oxford University students.

## Key Features

- **Scientific Mode**: Standard arithmetic plus advanced trigonometry, logs, exponentials.
- **Matrix Operations**: Add, multiply, invert, transpose, inverse. (Determinant available).
- **Graphing Tooling**: Responsive charting functions using Recharts and MathJS engine.

### Run locally:

```bash
npm install
npm run dev
```

### Stack

- React 18
- Vite
- TypeScript
- TailwindCSS
- MathJS
- Recharts (Charting library)
